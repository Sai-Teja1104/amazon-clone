# amazon-clone

This project is a static Amazon Clone website created to understand how large e-commerce websites are structured. It focuses mainly on UI design and layout, similar to Amazon’s homepage.

## Features

Amazon-like header with logo, search bar, and navigation

Product cards with images, price, and description

Banner / hero section

Footer similar to Amazon

Clean and simple UI

Beginner-friendly code structure

## Technologies used

HTML5

CSS3

(No JavaScript / Backend – UI only)
